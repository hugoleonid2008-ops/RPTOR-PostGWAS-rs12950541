# Data Files

This directory contains input and processed data files for the RPTOR post-GWAS functional annotation analysis.

## Directory Structure

```
data/
├── README.md                                      # This file
├── INPUT DATA (to be downloaded/generated)
│   ├── rnact_ENST00000306801_page1.csv           # RNAct predictions for RPTOR-201
│   ├── rnact_ENST00000574767_page1.csv           # RNAct predictions for RPTOR-208
│   ├── toppgene_ENST00000306801_enrichment.txt   # ToppGene enrichment for RPTOR-201
│   ├── toppgene_ENST00000574767_enrichment.txt   # ToppGene enrichment for RPTOR-208
│   ├── vep_rs12950541_ld_block.xlsx              # VEP annotations for LD block
│   ├── PheWAS_associations.csv                   # T2DKP PheWAS results for rs12950541
│   ├── clumped-variants_by_phenotype.csv         # Regional lead variants clumped by LD
│   └── associations_variants_in_region.csv       # Variant associations in 100kb window
└── PROCESSED DATA (generated by analysis scripts)
    └── See ../supplementary/ for output tables
```

## Input Data Sources

All data files should be sourced from public databases and tools. Below are instructions for obtaining each file.

### 1. RNAct Predictions

**Source:** https://rnact.crg.eu/

**RPTOR-201 (Canonical Transcript)**
- Ensembl ID: ENST00000306801
- Description: Canonical RPTOR protein-coding transcript
- Download Instructions:
  1. Visit https://rnact.crg.eu/
  2. Search for transcript ID: ENST00000306801
  3. Set prediction score filter: >= 1.0
  4. Download page 1 results as CSV
  5. Save as: `rnact_ENST00000306801_page1.csv`
- Expected Format: CSV with columns for protein ID, protein name, prediction score, etc.
- Expected Rows: ~100 protein interactions

**RPTOR-208 (NMD Isoform)**
- Ensembl ID: ENST00000574767
- Description: RPTOR isoform subject to nonsense-mediated decay
- Download Instructions:
  1. Visit https://rnact.crg.eu/
  2. Search for transcript ID: ENST00000574767
  3. Set prediction score filter: >= 1.5 (more stringent for shorter transcript)
  4. Download page 1 results as CSV
  5. Save as: `rnact_ENST00000574767_page1.csv`
- Expected Format: CSV with columns for protein ID, protein name, prediction score, etc.
- Expected Rows: ~100 protein interactions

### 2. ToppGene Enrichment Results

**Source:** https://toppgene.cchmc.org/

**RPTOR-201 Enrichment**
- Input: Protein list from rnact_ENST00000306801_page1.csv
- Upload Instructions:
  1. Visit https://toppgene.cchmc.org/
  2. Select "Gene Lists" tool
  3. Paste or upload protein list
  4. Run enrichment analysis with default parameters
  5. Download full results report
  6. Save as: `toppgene_ENST00000306801_enrichment.txt`
- Expected Content: Gene Ontology, pathway, and disease enrichment results

**RPTOR-208 Enrichment**
- Input: Protein list from rnact_ENST00000574767_page1.csv
- Instructions: Same as RPTOR-201, save as `toppgene_ENST00000574767_enrichment.txt`

### 3. Variant Effect Predictor Annotations

**Source:** Ensembl VEP release 105 (https://www.ensembl.org/Tools/VEP)

**VEP Analysis for rs12950541 LD Block**
- Variants: All variants in LD with rs12950541 (D' >= 0.7)
- Annotations: VEP predictions with CADD, SpliceAI, and dbNSFP plugins
- Generation Instructions:
  1. Run: `bash ../scripts/03_vep_analysis.sh`
  2. Output will be saved as: `vep_rs12950541_ld_block.xlsx` (after parsing/formatting)
- Expected Format: Excel spreadsheet with columns for:
  - Variant ID
  - Consequence type
  - IMPACT (HIGH/MODERATE/LOW)
  - CADD score
  - SpliceAI Delta score
  - dbNSFP predictions (BayesDel, ClinPred, DEOGEN2)
  - Regulatory features

### 4. T2D Knowledge Portal (T2DKP) Data

**Source:** https://t2d.hugeamp.org/

**PheWAS Associations**
- Source: T2DKP variant page for rs12950541
- Download Instructions:
  1. Visit https://t2d.hugeamp.org/variant.html?variant=rs12950541
  2. Scroll to "PheWAS Associations" section (bottom-line integrative analysis)
  3. Download CSV file with all phenotype-dataset combinations
  4. Save as: `PheWAS_associations.csv`
- Expected Format: CSV with columns for phenotype, P-value, beta/OR, sample size, dataset ID
- Expected Rows: 703 phenotype-dataset combinations tested, 78 significant at P < 0.05

**Regional Clumped Variants**
- Source: T2DKP region page (100kb window around rs12950541)
- Download Instructions:
  1. Visit https://t2d.hugeamp.org/region.html?chr=17&start=78684493&end=78784493&variant=17%3A78734493%3AG%3AA
  2. Download "Clumped Lead Variants" CSV grouped by phenotype
  3. Save as: `clumped-variants_by_phenotype.csv`
- Expected Format: CSV with columns for variant ID, phenotype, lead status, clumped variants
- Expected Rows: Lead variants per phenotype in region

**Regional Variant Associations**
- Source: T2DKP region page variant associations table
- Download Instructions:
  1. Same region URL as above
  2. Download "Variant Associations" table for all variants in 100kb window
  3. Save as: `associations_variants_in_region.csv`
- Expected Format: CSV with columns for variant ID, position, P-value, beta/OR, phenotype
- Expected Rows: 100-500 variants depending on LD structure

**Note on T2DKP Data:**
- Portal does not have a public programmatic API for data export
- All data must be downloaded manually via web interface
- "Bottom-line" integrative analysis: aggregates multiple GWAS datasets per phenotype
- Transcription factor binding motif predictions included in variant page

## Data Lineage and Processing

### Processing Pipeline

```
RNAct Queries (Manual)
  ↓
CSV files downloaded
  ↓
Protein lists extracted → ToppGene (Manual)
  ↓
Enrichment results
  ↓
Supplementary Tables S1-S4
```

```
Ensembl REST API
  ↓
LD partners retrieved
  ↓
VEP Analysis (Automated)
  ↓
VEP output parsed
  ↓
Supplementary Tables S2
```

### Data Quality Notes

- **RNAct Data:** Raw predictions from catRAPID algorithm. Score thresholds chosen based on transcript length.
- **ToppGene Data:** Standard enrichment analysis with multiple testing correction applied.
- **VEP Data:** Variant consequences annotated against Ensembl genome build GRCh38. Regulatory regions from Ensembl regulatory build.
